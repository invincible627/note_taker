# Notebook
A graphic notebook with Python that can also save files in txt format
## Languages available
english and persian
### Libraries used for this program
wxpython, json
#### How to run
Install the wxpython library, put the languages next to the .py file, and run the .py file.
#### note_taker capabilities updated by AmirMahdi Fard

Added ability to rename tabs
Added settings menu
Added the ability to import a new file/tab

# دفترچه یادداشت
یک دفترچه ی یادداشت گرافیکی با پایتون که قابلیت ذخیره ی فایل رو هم داره به فرمت txt
## زبانهای در دسترس
انگلیسی و فارسی
### کتابخانه های مورد استفاده برای این برنامه
wxpython, json
#### نحوه اجرا
کتابخانه ی wxpython رو نصب کنید، زبانها کنار فایل .py قرار داشته باشن، و فایل .py رو اجرا کنید
#### قابلیتهای note_taker بروز توسط امیر مهدی فرد
دارای یک منو
اضافه شدن قابلیت انتخاب زبان
اضافه شدن قابلیت تغییر اسم تبها
اضافه شدن منوی تنظیمات
اضافه شدن قابلیت ایمپورت فایل/تب جدید
