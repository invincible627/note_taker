# note_taker
A graphic note taker with Python that has the ability to save files in txt format 
## availeble languages
english and persian
### Libraries used for this program
wxpython, json
#### how to run
Install the wxpython library, put the languages ​​next to the .py file, and run the .py file.
#### Updated note_taker capabilities
Has a menu
Added language selection feature
Added ability to rename tabs
Added settings menu
Added the ability to import a new file/tab
